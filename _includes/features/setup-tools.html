Configuration built for Leap 16

openSUSE Leap 16 introduces Myrlyn and Cockpit as the primary setup and system management experience. These modern tools guide new users through post-install configuration while giving administrators powerful, browser-based controls.

Myrlyn
  • Welcome wizard for first-boot personalization
  • Guided hardware, localization, and account setup
  • Extensible modules for desktop and server roles
Cockpit
  • Graphical management for services, storage, and updates
  • Works locally or remotely from any browser
  • Integrates with systemd, transactional updates, and containers
